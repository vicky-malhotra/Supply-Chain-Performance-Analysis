# Supply-Chain-Performance-Analysis
This project involved a thorough supply chain analysis for a Fashion and Beauty startup, concentrating on the entire lifecycle of makeup products. The effort demonstrates how data interpretation informs strategy and operational efficiency. The rich dataset contains critical features—including Production Volumes, Lead Times, Shipping Costs, Defect Rates, and Customer Demographics—all vital for understanding product flow from supplier to customer.
